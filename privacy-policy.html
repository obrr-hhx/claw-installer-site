Claw Installer 隐私政策

最后更新：2026 年 3 月 3 日

概述

Claw Installer 是一款 macOS 菜单栏应用，用于管理和访问本地运行的 OpenClaw AI 网关。我们高度重视您的隐私。

数据收集

Claw Installer 不收集、不存储、不传输任何个人数据。

  • App 不包含任何分析工具或追踪代码
  • App 不联网到任何远程服务器（仅连接本机 127.0.0.1 上的 OpenClaw 服务）
  • 您配置的 API Key 仅保存在您的 Mac 本地（~/.openclaw/ 目录），不会上传至任何第三方

网络连接

Claw Installer 仅与运行在您本机（127.0.0.1）的 OpenClaw 网关通信。App 本身不发起任何外部网络请求。

当您通过 OpenClaw Web UI 与 AI 模型交互时，数据由 OpenClaw 直接发送至您配置的 AI 提供商（如 Anthropic、OpenAI 等），Claw Installer 不参与该过程。

本地存储

Claw Installer 在您的 Mac 上存储以下数据：

  • App 偏好设置（端口号等），存储在 macOS 标准 UserDefaults 中
  • AI 模型配置和 API Key，存储在 ~/.openclaw/ 目录中（OpenClaw 的标准配置目录）

第三方服务

Claw Installer 不集成任何第三方 SDK、分析工具或广告服务。

Privacy Policy

Last updated: March 3, 2026

Overview

Claw Installer is a macOS menu bar app for managing and accessing the locally-running OpenClaw AI gateway. We take your privacy seriously.

Data Collection

Claw Installer does not collect, store, or transmit any personal data.

  • The app contains no analytics or tracking code
  • The app does not connect to any remote servers (it only connects to OpenClaw running on localhost 127.0.0.1)
  • Your API keys are stored locally on your Mac (~/.openclaw/ directory) and are never uploaded to any third party

Network Connections

Claw Installer only communicates with the OpenClaw gateway running on your local machine (127.0.0.1). The app itself does not make any external network requests.

When you interact with AI models through the OpenClaw Web UI, data is sent directly from OpenClaw to your configured AI provider (e.g., Anthropic, OpenAI). Claw Installer is not involved in this process.

Local Storage

Claw Installer stores the following data on your Mac:

  • App preferences (port number, etc.) in macOS standard UserDefaults
  • AI model configuration and API keys in ~/.openclaw/ (OpenClaw's standard config directory)

Third-Party Services

Claw Installer does not integrate any third-party SDKs, analytics, or advertising services.
